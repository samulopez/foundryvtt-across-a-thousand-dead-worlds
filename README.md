# Across a Thousand Dead Worlds for Foundry VTT

**foundryvtt-across-a-thousand-dead-worlds** is an unofficial system for playing [Across a Thousand Dead Worlds](https://blackoathgames.com/across-a-thousand-dead-worlds) on [Foundry VTT](http://foundryvtt.com/)

## Features

- Character sheets
- Creature sheets
- NPC sheets

## Installation

Install the system from within Foundry (Configuration and Setup, Game Systems tab, Install System button). Find the package Across a Thousand Dead Worlds and install.

Alternatively, you can manually install the system by following these steps:

1. Inside Foundry, select the Game Systems tab in the Configuration and Setup menu.
2. Click the Install System button and enter in the Manifest URL the following URL: `https://github.com/samulopez/foundryvtt-across-a-thousand-dead-worlds/releases/latest/download/system.json`

3. Click Install and wait for installation to complete.
